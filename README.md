# INCD-2025
Industrial Network Component Dataset 2025

This repository is created in order to store the dataset used for the development of Master Thesis "HYBRID LSTM-CNN FRAMEWORK FOR PREDICTIVE ANOMALY DETECTION IN OPERATIONAL TECHNOLOGY NETWORKS". The dataset is in long format and needs filtering.

Work was submitted in 11.2025, Tampere University.

The work is planned to be extended by research paper. The code will be uploaded to this repository.
